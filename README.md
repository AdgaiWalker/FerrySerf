# 奴书（串行版）
## 前言
**在这里,你就是皇帝,你有着无比忠诚的大臣与奴隶，你的意愿就是他们的行动指向标。**

**根本目的**：降低使用成本，省token，大众可承担。

**核心技术**：协议（权力）层握手。比如软件之间都有终端最高权限，他们就可以相互指挥；最高权限可以向下兼容，终端协议指挥应用层协议，应用层不能指挥终端层，比如cli可以操控网页，网页却不能操控终端。
基于opencalw、IDE，三层分工协议
token均衡分布，整体性减少AI上下文过载消耗token，达到Ferry计划的可落地性目标的关键一环
